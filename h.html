Krzywa Hilberta

Krzywą wynalazł David Hilbert w roku 1891.

Po głębszej analizie można się zorientować, że krzywą Hi+1 otrzymano ze złożenia 4 krzywych Hi odpowiednio obróconych i powiązanych razem trzema liniami. Zauważmy, że H1 może być traktowana jako złożenie 4 pustych krzywych H0, powiązanych ze sobą trzema liniami prostymi.

Ponieważ każda krzywa Hi składa się z 4 dwukrotnie zmniejszonych kopii krzywej Hi-1, wydaje się naturalne podzielenie procedury rysującej Hi na cztery części, z których każda rysuje krzywą Hi-1 odpowiednio obróconą. Jeśli te cztery części oznaczymy przez A, B, C i D, a procedury kreślące linie łączące będziemy oznaczać strzałkami odpowiednio zorientowanymi, to uzyskamy następujący schemat rekurencji:
A: D←A↓A→B
B: C↑B→B↓A
C: B→C↑C←D
D: A↓D←D↑C

Poniżej zaprezentowano kod w języku JavaScript kreślący widoczną na płótnie krzywą Hilberta. Program — względem powyższej prezentacji — uproszczono o ustawiane kolorów, centrowanie, i ustalanie odstępów. Niech h będzie jednostkową długością odcinka:

var c = document.getElementById("canvas"),
    stopien = 2,
    x = 0,
    y = 0;
A(stopien);
function A(st) {
	if (st == 0) return;
	D(st - 1); c.moveTo(x, y); x -= h; c.lineTo(x, y); 
	A(st - 1); c.moveTo(x, y); y -= h; c.lineTo(x, y); 
	A(st - 1); c.moveTo(x, y); x += h; c.lineTo(x, y); 
	B(st - 1);
}
function B(st) {
	if (st == 0) return;
	C(st - 1); c.moveTo(x, y); y += h; c.lineTo(x, y); 
	B(st - 1); c.moveTo(x, y); x += h; c.lineTo(x, y); 
	B(st - 1); c.moveTo(x, y); y -= h; c.lineTo(x, y); 
	A(st - 1);
}
function C(st) {
	if (st == 0) return;
	B(st - 1); c.moveTo(x, y); x += h; c.lineTo(x, y);  
	C(st - 1); c.moveTo(x, y); y += h; c.lineTo(x, y); 
	C(st - 1); c.moveTo(x, y); x -= h; c.lineTo(x, y); 
	D(st - 1);
}
function D(st) {
	if (st == 0) return;
	A(st - 1); c.moveTo(x, y); y -= h; c.lineTo(x, y);
	D(st - 1); c.moveTo(x, y); x -= h; c.lineTo(x, y); 
	D(st - 1); c.moveTo(x, y); y += h; c.lineTo(x, y); 
	C(st - 1);
}
c.stroke();
